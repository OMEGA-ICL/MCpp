Detailed Description

template<class T>
class mc::PolImg< T >

mc::PolImg is a C++ class for evaluation of the polytoptic image of a factorable function. Propagation of the image is via polytopic arithmetic, as implemented in mc::PolVar. The template parameter corresponds to the type used to propagate variable range. Round-off errors are not accounted for in the computations (non-verified implementation).
