Detailed Description

template<typename T>
class mc::SCModel< T >

mc::SCModel is a C++ class for definition of Chebyshev model environment in sparse format. Propagation of Chebyshev models for factorable functions is via the C++ class mc::SCVar. The template parameter corresponds to the type used to propagate the remainder bound.
